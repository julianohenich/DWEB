Projeto na Trilha do Aprendizado

HTML é uma linguagem de marcação de texto para criar sites web
Para começar temos a estrutura de um html.
Com o head que é o cabeçalho do site onde coloca o titulo usando title, que é o nome que aparece na guia do browser e também aparece o icone que é adicionando usando link rel.
Depois tem o body que é o corpo do site, onde vai toda informação.
Temos a tag h que vai de 1 a 6, que é referente a titulos com alterações de tamanho
Tag p para paragráfos
Tags b, strong, i, em, para negrito, texto destacado, itálico e ênfase respectivamente
Colocamos imagens no html usando img search="local"
Fazemos hiperlinks externos, internos e locais com a href="" e também arquivos para dowloads
